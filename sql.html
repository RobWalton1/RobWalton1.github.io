NodeJS Project

As a part of my third-year project at university, I developed a low-cost inventory management system using NodeJS and a MySQL database. Tailored for warehouse environments, the system efficiently tracks, manages, and analyzes inventory. It features a web-based application built on NodeJS, while the MySQL database, designed meticulously, handles inventory items, suppliers, orders, and transactions. The project's emphasis on SQL is evident in optimized queries that provide insightful reports on stock, sales trends, and more. This project showcases my skills in software development, database design, and data analysis, all applied to a real-world scenario.
